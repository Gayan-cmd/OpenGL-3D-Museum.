# OpenGL-3D-Museum.
Interactive 3D Science Museum built with C++ and OpenGL/GLUT.
<img width="1918" height="992" alt="Museum" src="https://github.com/user-attachments/assets/84fdc328-d79d-478a-a8ed-85fdbc0931a7" />
A 3D virtual environment created using C++, OpenGL, and GLUT. Features include:
- Holographic Solar System with shader-like glitch effects.
- Different Artifacts Modeled Using OpenGL
- Procedural Particle effects (Lightning).
- Texture mapping using SOIL.
